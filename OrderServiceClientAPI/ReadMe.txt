OrderServiceAPI
================
Create an order hit this endpoint - pass in orderDTOcreate.cs - returns orderDTO
localhost:port/order - post

Get Order by orderid - returns orderDTO
localhost:port/order/orderid/{orderid} - get

Get Order by userid - returns orderDTO
localhost:port/order/userid/{userid} - get

Delete order by orderid
localhost:port/order/{orderid} - delete
